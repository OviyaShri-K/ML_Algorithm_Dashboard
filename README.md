# 🧠 ML Algorithm Dashboard

An interactive Machine Learning Dashboard built using Python, Streamlit, Pandas, and Scikit-Learn. This dashboard demonstrates both Supervised and Unsupervised Machine Learning Algorithms using sample datasets with real-time predictions and clustering.

---

## 📌 Features

- Interactive Streamlit Dashboard
- Dataset Preview
- Model Training
- Model Evaluation
- User Input Prediction
- Supervised & Unsupervised Learning Algorithms
- Simple and User-Friendly Interface

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-Learn

---

## 📚 Machine Learning Algorithms

### Supervised Learning

-  Linear Regression
-  Logistic Regression
-  Decision Tree
-  Random Forest
-  K-Nearest Neighbors (KNN)

### Unsupervised Learning

-  K-Means Clustering
-  DBSCAN
-  Hierarchical Clustering
-  Principal Component Analysis (PCA)
-  Mean Shift

## ⚙️ Installation



Go to the project folder

```bash
cd ML_Algorithm_Dashboard
```

Install required packages

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
